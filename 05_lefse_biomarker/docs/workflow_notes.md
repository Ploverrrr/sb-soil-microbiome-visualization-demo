# Workflow Notes

This module implements the LEfSe biomarker workflow with `microeco`, matching a reference analysis and plotting structure while using only simulated shared toy data.

## Analysis Steps

1. Read shared sample metadata, abundance table, and taxonomy table.
2. Check that required files and columns exist.
3. Check that sample IDs match between metadata and abundance table.
4. Check that feature IDs match between abundance and taxonomy tables.
5. Convert feature counts to within-sample relative abundance for transparent intermediate output.
6. Add standard taxonomy prefixes internally (`k__`, `p__`, `c__`, `o__`, `f__`, `g__`) before building the `microeco` taxonomy table.
7. Build a `microeco::microtable` from the raw-like count table, sample metadata, and taxonomy table.
8. Run `microeco::trans_diff$new()` with:

```r
method = "lefse"
group = "Group"
taxa_level = "all"
alpha = 0.05
P_adjust_method = "fdr"
lefse_norm = 1000000
```

9. Export `res_diff` as the recalculated LEfSe biomarker table.
10. Use a cloned LEfSe model restricted to the top plotted features for the LDA score barplot.
11. Draw the LDA score plot with `plot_diff_bar()`.
12. Draw the native LEfSe cladogram with `plot_diff_cladogram()`.
13. Draw the KW abundance plot with `plot_diff_abund()` using the same selected taxa as the LDA plot.
14. Combine LDA + cladogram with `aplot::insert_left()` when `aplot` is available, otherwise use a `patchwork` fallback.
15. Combine LDA + KW abundance with `gridExtra::arrangeGrob()`.
16. Export model-derived plotting tables to `results/`.
17. Save PDF and PNG figures to `figures/`.

## Interpretation

The LDA barplot ranks taxa by their LEfSe effect size. The color indicates the group in which each biomarker is enriched.

The cladogram is generated by `microeco::plot_diff_cladogram()`, not by a hand-written approximation. Rings represent taxonomic hierarchy, colored nodes indicate group-enriched biomarkers, and letter annotations identify selected clades in the plot legend.

The KW abundance plot uses `microeco::plot_diff_abund()` to show group abundance patterns for the same selected taxa used in the LDA barplot.

The auxiliary heatmap summarizes model-derived group mean abundance for selected biomarkers. It is included as a portfolio-friendly companion view, not as a replacement for the LEfSe-style demo outputs.

## Scope

This module is intended for visualization portfolio demonstration. It uses simulated/desensitized toy data and should not be interpreted as a real biological result. For a formal analysis, users should verify assumptions, sequencing-depth handling, multiple-testing choices, and LEfSe settings for their own dataset.
