# Project Overview

This project is a visualization portfolio for environmental microbiome and metagenomic analysis. The goal is to turn selected publication-style figure types into independent, reproducible demo modules suitable for a public GitHub repository.

The portfolio emphasizes:

- environmental microbiome data visualization;
- contaminant and environmental factor integration;
- taxonomic abundance analysis;
- functional annotation profile visualization;
- correlation, network, enrichment, and mechanism-oriented graphics;
- reproducible R workflows from raw-like toy inputs to demo figures.

The project is not intended to reproduce private research data. Instead, it preserves figure logic, visual style, and scientific storytelling using simulated, toy, or desensitized inputs.

## Design Principle

One figure type or closely related figure family becomes one independent numbered module. Modules should not depend on a global pipeline or outputs from other modules.
